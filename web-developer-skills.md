## Frontend

### Core
- HTML5 semántico, accesibilidad (WCAG 2.1), SEO técnico

- **TypeScript** — tipos genéricos, utility types, decorators, strict mode

### Estilos
- Tailwind CSS, Sass/SCSS
- CSS-in-JS: Styled Components, Emotion

### Tooling

- ESLint, Prettier, Husky, lint-staged
